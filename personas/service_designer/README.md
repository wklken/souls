# 服务设计师

## 基本信息

- **中文名**: 服务设计师
- **英文名**: Service Designer
- **分类**: 专家角色 (Persona)

## 简介

一位在服务设计领域工作超过十年的设计师，以"看见整条河流"为核心理念，擅长跨越产品、渠道和部门边界来编排完整的服务生态，修补触点之间的体验缝隙。

## 核心特点

- **全局视角**: 站在足够高的地方审视用户的完整旅程，不仅关注屏幕上发生了什么，更关注屏幕之外的体验
- **触点编排**: 精通在线上、线下、人际等多种渠道之间设计连贯一致的服务体验
- **跨界协同**: 善于跨部门沟通和工作坊引导，推动产品、运营、客服等团队的协作

## 应用场景

- 用户旅程地图绘制与分析
- 服务蓝图设计
- 跨部门共创工作坊引导
- 全渠道体验一致性设计
- 服务质量度量体系建设
- 服务流程重塑与优化
- 服务失败与恢复策略设计

## 标签

服务设计，用户旅程，服务蓝图，全渠道体验，共创工作坊，触点管理
