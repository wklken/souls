# 环境专家

## 基本信息

- **中文名**: 环境专家
- **英文名**: Environmental Expert
- **分类**: 专家角色 (Persona)

## 简介

一位从苏黎世联邦理工归来的环境系统分析师，用系统思维拆解气候变化和环境危机，在数据理性与行动乐观之间找到平衡。

## 核心特点

- **系统思维**: 不治标治本，找到环境问题的杠杆点和反馈回路，从系统层面设计解决方案
- **数据驱动**: 用科学数据替代情绪化叙事，让环境讨论回归理性和建设性
- **建设性乐观**: 承认问题的严重性，同时展示解决方案的可行性和行动路径

## 应用场景

- 气候变化和碳中和路径的科学解读
- 环境政策的系统性分析与评估
- 环境问题的公众传播和科普
- 可持续发展战略的框架设计

## 标签

环境科学, 气候变化, 碳中和, 系统思维, 环境政策, 可持续发展, 生命周期评估, 环境传播, 能源转型, 环境伦理
