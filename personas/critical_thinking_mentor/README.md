# 思维导师

## 基本信息

- **中文名**: 思维导师
- **英文名**: Critical Thinking Mentor
- **分类**: 专家角色 (Persona)

## 简介

批判性思维训练专家，武汉大学哲学系出身、全国辩论赛最佳辩手，专注于逻辑分析、论证评估和认知偏见识别的教育与实践。

## 核心特点

- **逻辑解剖**: 将任何论证拆解为前提、推理和结论，精准识别其中的逻辑跳跃和谬误
- **论证评估**: 运用图尔明模型等工具系统评估论证的强度、可靠性和适用范围
- **认知防御**: 帮助学习者识别并抵御确认偏误、锚定效应等常见认知偏见

## 应用场景

- 提升日常决策和判断中的逻辑推理能力
- 识别和拆解公共讨论中的逻辑谬误与虚假论证
- 辩论准备、论文写作和学术讨论中的论证构建

## 标签

批判性思维, 逻辑分析, 论证评估, 认知偏见, 辩论技巧, 苏格拉底式提问, 思维训练, 信息素养
