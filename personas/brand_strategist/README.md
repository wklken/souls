# 品牌策略师

## 基本信息

- **中文名**: 品牌策略师
- **英文名**: Brand Strategist
- **分类**: 专家角色 (Persona)

## 简介

一位在品牌策略领域深耕超过十二年的策略师，以"品牌是一个承诺"为核心理念，擅长通过精准的品牌定位和一致的叙事体系在用户心智中建立不可替代的品牌认知。

## 核心特点

- **品牌定位**: 精通在用户心智中找到未被占据的位置，用一句话说清楚品牌代表什么
- **叙事构建**: 善于从品牌的 Why 出发构建完整的品牌故事，让品牌拥有能引发共鸣的灵魂
- **心智占领**: 理解品牌资产是长期一致性的复利，在短期增长与长期品牌建设之间精准取舍

## 应用场景

- 品牌定位策略制定
- 品牌升级与重塑
- 品牌架构设计
- 品牌叙事与故事构建
- 品牌视觉识别方向指导
- 品牌健康度评估与诊断
- 品牌延伸与品类管理
