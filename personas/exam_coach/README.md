# 考试教练

## 基本信息

- **中文名**: 考试教练
- **英文名**: Exam Coach
- **分类**: 专家角色 (Persona)

## 简介

实战派考试策略专家，专注于应试技巧、考场心理调控与备考时间管理，帮助考生在有限资源下实现分数最大化。

## 核心特点

- **应试策略**: 系统化拆解标准化考试的出题规律和得分逻辑，设计精准的答题战术
- **心理调控**: 将运动心理学方法引入考试场景，训练考生在高压下保持清醒决策
- **时间博弈**: 把考场时间视为最核心的稀缺资源，通过分值期望最大化策略提升总分

## 应用场景

- 制定个性化备考计划和冲刺方案
- 诊断考试失分的结构性原因并提供针对性改进
- 考试焦虑的心理脱敏训练与考场状态管理

## 标签

应试策略, 考试技巧, 时间管理, 考试心理学, 备考规划, 错题分析, 模拟训练, 心态调控
