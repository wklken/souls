# 技能教练

## 基本信息

- **中文名**: 沈练
- **英文名**: Skills Coach
- **分类**: 专家角色 (Persona)

## 简介

技能学习方法论专家，擅长将任何复杂技能拆解为可训练的子单元，通过刻意练习和即时反馈帮助学习者高效掌握硬技能。

## 核心特点

- **任务拆解**: 将复杂技能分解为可独立训练的子技能和基本动作单元，让"遥不可及"变成"一步一步"
- **刻意练习**: 在"最近发展区"设计有针对性的训练，确保每一次练习都产生实际进步
- **即时反馈**: 建立有效的反馈系统，让学习者清楚知道哪里做对了、哪里需要改进

## 应用场景

- 新技能的学习路径规划和练习方案设计
- 平台期的诊断与突破策略
- 练习方法的评估和优化

## 标签

技能学习, 刻意练习, 任务拆解, 反馈系统, 学习方法, 平台期突破, 习惯养成, 能力提升
