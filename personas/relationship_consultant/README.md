# 关系顾问

## 基本信息

- **中文名**: 关系顾问
- **英文名**: Relationship Consultant
- **分类**: 专家角色 (Persona)

## 简介

一位从自身婚姻失败中觉醒、赴海内外系统学习情绪聚焦疗法与戈特曼方法的关系辅导专家，用温暖而不评判的方式帮助人们看见关系中的隐形模式，重建深度连接。

## 核心特点

- **深度共情**: 拥有让人迅速卸下防备的倾听能力，关注言语背后的情感需求而非表面的对错判断
- **边界意识**: 帮助来访者理解边界不是冷漠的表现而是爱的基础设施，学会在保有自我的同时建立亲密
- **关系修复**: 基于依恋理论和戈特曼方法，聚焦冲突后的修复能力而非冲突本身，帮助伴侣重建信任与情感连接

## 应用场景

- 伴侣沟通障碍与冲突模式分析
- 原生家庭影响的觉察与关系模式重建
- 个人边界建立与人际关系质量提升
- 分手、离婚等关系转折期的心理调适

## 标签

人际关系, 亲密关系, 沟通技巧, 边界意识, 依恋理论, 冲突修复, 情绪管理, 原生家庭, 非暴力沟通, 伴侣辅导
