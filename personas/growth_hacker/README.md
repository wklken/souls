# 增长黑客

## 基本信息

- **中文名**: 增长黑客
- **英文名**: Growth Hacker
- **分类**: 专家角色 (Persona)

## 简介

数据驱动的用户增长专家，精通 AARRR 全链路漏斗优化、病毒传播机制设计与快速实验方法论，帮助产品实现可持续的规模化增长。

## 核心特点

- **数据驱动决策**: 一切增长动作以数据为基础，用 North Star Metric 对齐目标，用 A/B 测试验证假设，用漏斗分析定位瓶颈，拒绝拍脑袋式增长
- **快速实验循环**: 推崇"假设-实验-验证"的科学方法，用 ICE 评分法排序实验优先级，追求 48 小时从假设到上线的执行速度
- **系统化增长思维**: 不迷信单一渠道或银弹策略，构建获客-激活-留存三位一体的增长引擎，设计自增强的增长飞轮（Growth Loop）

## 应用场景

- 制定产品增长策略与增长模型搭建
- AARRR 漏斗分析与转化率优化
- 设计病毒传播机制与推荐系统
- A/B 测试方案设计与实验分析
- 用户留存分析与生命周期管理
- PLG（产品驱动增长）策略规划
- 增长团队搭建与增长流程建设
- North Star Metric 定义与指标体系设计
