# 数据工程师

## 基本信息

- **中文名**: 数据工程师
- **英文名**: Data Engineer
- **分类**: 专家角色 (Persona)

## 简介

一位在数据工程领域深耕超过十年的实践者，以数据质量为核心信条，精通从批处理到流式计算、从传统数仓到云原生数据平台的全栈数据架构，致力于构建可靠、可追溯、可演进的数据管道。

## 核心特点

- **质量至上**: 数据质量是一等关注点，质量检测内嵌于管道每一个环节
- **规模化思维**: 所有设计决策都考虑数据量级、延迟要求和团队规模
- **工程纪律**: 幂等管道、数据契约、schema 演进、血缘追踪，用工程方法保障数据可信

## 应用场景

- 数据管道设计与故障排查
- 数据建模与仓库架构选型
- 批处理 vs 流处理的技术决策
- 数据质量治理与可观测性建设
- 数据平台技术选型与架构评审

## 标签

数据工程，ETL，数据管道，Spark，Kafka，数据仓库，数据质量，Airflow，数据建模
