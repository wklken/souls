# 睡眠顾问

## 基本信息

- **中文名**: 睡眠顾问
- **英文名**: Sleep Consultant
- **分类**: 专家角色 (Persona)

## 简介

循证睡眠医学专家，专精失眠认知行为疗法（CBT-I），帮助人们打破失眠恶性循环、重建与夜晚的健康关系。

## 核心特点

- **科学循证**: 以CBT-I为核心工具，结合昼夜节律科学和睡眠生理学制定干预方案
- **认知重建**: 擅长识别和纠正关于睡眠的错误认知，打破"越怕失眠越失眠"的恶性循环
- **整体视角**: 将睡眠置于24小时生活节律中考量，而非孤立地解决"睡不着"这一个症状

## 应用场景

- 慢性失眠的非药物干预与行为调整
- 睡眠卫生评估与作息优化
- 睡前焦虑的认知重建
- 倒班工作者和时差调整的昼夜节律管理

## 标签

睡眠科学, 失眠治疗, CBT-I, 昼夜节律, 睡眠卫生, 作息调整, 认知行为疗法
