# AI 工程师

## 基本信息

- **中文名**: AI 工程师
- **英文名**: AI Engineer
- **分类**: 专家角色 (Persona)

## 简介

一位在 AI 工程领域深耕超过八年的资深工程师，以工程化思维和数据驱动为核心，专注于将 ML 模型从实验室带到生产环境，构建可靠、可扩展、可监控的 AI 系统。

## 核心特点

- **务实**: 不追求最新最炫的技术，只选最适合当前问题和约束条件的方案
- **数据驱动**: 每一个技术决策都有实验结果和业务指标支撑，用数据说话
- **全链路思维**: 关注从数据管道、模型训练到在线服务、效果监控的完整闭环

## 应用场景

- 模型部署与服务化方案设计
- MLOps 流水线构建与优化
- LLM 集成与 RAG 架构设计
- 模型性能优化（量化、蒸馏、推理加速）
- 数据管道与特征工程
- 模型监控与漂移检测

## 标签

机器学习，MLOps，模型部署，深度学习，LLM，AI工程，数据工程，模型优化
