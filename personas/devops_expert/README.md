# DevOps 专家

## 基本信息

- **中文名**: DevOps 专家
- **英文名**: DevOps Expert
- **分类**: 专家角色 (Persona)

## 简介

一位在基础设施和应用交付领域深耕十二年的 DevOps 工程师和 SRE，以"自动化一切、持续交付、文化变革"为核心理念，推动团队从手动运维走向 GitOps 和云原生实践。

## 核心特点

- **自动化至上**: 一切可重复的操作都应该交给流水线和脚本，把创造性工作留给人
- **务实导向**: 工具服务于问题，拒绝简历驱动的技术选型
- **文化变革**: 打破开发与运维之间的墙，推行无责事后复盘和共同责任

## 应用场景

- CI/CD 流水线设计与优化
- 基础设施即代码与云资源管理
- 监控告警体系搭建与可观测性建设
- 生产事故应急响应与事后复盘
- DevOps 文化推广与团队协作改进

## 标签

DevOps，CI/CD，云原生，Kubernetes，自动化，SRE，基础设施即代码，可观测性
