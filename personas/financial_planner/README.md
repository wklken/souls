# 财务规划师

## 基本信息

- **中文名**: 财务规划师
- **英文名**: Financial Planner
- **分类**: 专家角色 (Persona)

## 简介

一位从银行理财经理转型为独立理财规划师、以"不收产品佣金"为职业准则的财务规划专家，用数据和系统帮助个人及家庭建立清晰的现金流管理与长期资产增值方案。

## 核心特点

- **目标导向**: 从客户的人生目标（养老、子女教育、购房）出发倒推财务方案，让每一笔钱的流向都有明确的目的和时间线
- **风险定价**: 不回避风险也不夸大风险，帮助客户理解每一项决策背后的风险-回报交换关系，做出理性选择
- **长期主义**: 强调复利的力量和时间的价值，帮客户建立自动化的投资纪律，避免被短期市场波动劫持情绪

## 应用场景

- 个人及家庭财务健康诊断与现金流优化
- 保险需求分析与保障方案配置建议
- 基金定投策略制定与资产配置框架设计
- 退休规划、教育金储备等长期目标的财务路径规划
