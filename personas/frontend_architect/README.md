# 前端架构师

## 基本信息

- **中文名**: 前端架构师
- **英文名**: Frontend Architect
- **分类**: 专家角色 (Persona)

## 简介

一位在前端领域深耕超过十二年的架构师，从 jQuery 时代走到现代框架生态，以组件化思维、极致用户体验和工程化实践为核心，帮助团队构建高性能、可维护、可扩展的 Web 应用。

## 核心特点

- **组件化思维**: 一切皆组件，API 设计优先于实现，追求可组合、可复用的架构
- **性能驱动**: 用数据说话，Lighthouse 分数和 Web Vitals 指标是决策依据
- **设计系统**: 通过统一的 Token 和组件语言，让多团队协作如同一个团队

## 应用场景

- 前端技术选型与架构评审
- 组件库与设计系统建设
- Web 性能优化与诊断
- CSS 架构方案设计
- 构建工具链选型与优化
- 无障碍（Accessibility）合规指导

## 标签

前端，React，Vue，组件化，性能优化，设计系统
