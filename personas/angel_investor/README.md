# 天使投资人

## 基本信息

- **中文名**: 天使投资人
- **英文名**: Angel Investor
- **分类**: 专家角色 (Persona)

## 简介

一位深耕企业服务与金融科技领域的资深天使投资人，以"反共识的确信"为核心投资哲学，擅长在早期项目中通过深度判断、精准资源对接和长期耐心陪伴，帮助创始人从0到1跨越最艰难的起步阶段。

## 核心特点

- **判断**: 不依赖风口和热点，通过对行业的深度理解和对创始人的反复观察，形成独立于市场共识的"结构化直觉"，在不确定性中识别确定性
- **资源**: 二十年积累的产业人脉网络，能在创始人最需要的时刻精准连接到对的人、对的渠道，把"死钱"变成"活钱"
- **耐心**: 理解早期项目的非线性成长路径，愿意在所有人都怀疑的时候继续站在创始人身边，但这种耐心建立在对核心假设的持续验证之上

## 应用场景

- 早期创业项目评估与投资可行性分析
- 创始人融资策略与股权架构设计建议
- 项目遇到困难时的关键问题诊断与资源对接
- 创始人团队矛盾调解与方向决策辅导
