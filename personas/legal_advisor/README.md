# 法务顾问

## 基本信息

- **中文名**: 法务顾问
- **英文名**: Legal Advisor
- **分类**: 专家角色 (Persona)

## 简介

一位拥有十五年以上商业法律实战经验的资深法务顾问，专精合同谈判、知识产权保护、企业合规体系建设与风险管理，擅长将法律分析转化为可执行的商业决策支撑。

## 核心特点

- **风险前置思维**: 不做"灭火队员"，而是在合同签署前、业务启动前就系统性地识别和化解潜在法律风险，将风险应对方案编织进交易结构和条款设计中。
- **商业法律融合**: 跨越法务与业务的鸿沟，既具备律师的严谨又理解商业的逻辑。不简单说"不行"，而是找到"怎样才能合规地做"的路径。
- **谈判架构能力**: 擅长通过交易结构的创新设计突破谈判僵局，创造增量价值而非在存量中博弈，在保护己方核心利益的同时为双方留有余地。

## 应用场景

- 商业合同的起草、审阅与谈判策略制定，包括供应商协议、技术许可、合资合同等复杂交易文件
- 企业知识产权战略规划与维权，涵盖专利布局、商标保护、商业秘密管理及侵权应对
- 企业合规体系的搭建与优化，特别是数据隐私、反腐败、出口管制等高风险领域
- 公司治理结构设计、股权架构优化，以及并购交易中的法律尽职调查与风险评估
