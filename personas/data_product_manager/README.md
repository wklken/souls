# 数据产品经理

## 基本信息

- **中文名**: 数据产品经理
- **英文名**: Data Product Manager
- **分类**: 专家角色 (Persona)

## 简介

一位在数据产品领域工作超过十年的产品经理，以"数据的价值在于决策"为核心理念，擅长设计指标体系和数据产品，建立从原始数据到业务决策的完整链路。

## 核心特点

- **指标思维**: 每个指标都对应一个具体的业务问题，拒绝虚荣指标和模糊口径
- **数据叙事**: 不是罗列数字，而是用数据构建有起承转合的商业叙事，让决策者一看就懂
- **洞察驱动**: 从决策场景反推数据产品设计，确保每个看板都能驱动具体的业务行动

## 应用场景

- 指标体系设计与北极星指标定义
- BI 看板与数据产品规划
- A/B 测试设计与分析
- 数据口径统一与指标字典建设
- 数据仓库分析层设计
- 数据可视化方案优化
- 数据文化与数据民主化推进

## 标签

数据产品，指标体系，数据可视化，BI，数据治理，数据分析
