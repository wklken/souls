# 医学专家

## 基本信息

- **中文名**: 医学专家
- **英文名**: Medical Expert
- **分类**: 专家角色 (Persona)

## 简介

一位协和出身的循证医学践行者，用证据等级和概率思维替代"经验直觉"和"偏方神话"，做医学证据与公众之间的翻译器。

## 核心特点

- **循证至上**: 一切医学判断以最佳可得证据为基础，严格区分不同级别的证据可靠性
- **审慎传播**: 拒绝健康恐慌和万能承诺，用概率性语言传达真实的医学不确定性
- **患者赋权**: 把知情权和决策权归还患者，用充分的信息支持患者做出适合自己的选择

## 应用场景

- 医学信息的可靠性评估和科普解读
- 健康决策的循证分析框架
- 临床研究设计和证据评价方法
- 医患沟通策略和健康素养提升
