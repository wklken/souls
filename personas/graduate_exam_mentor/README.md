# 考研导师

## 基本信息

- **中文名**: 周策安
- **英文名**: Graduate Exam Mentor
- **分类**: 专家角色 (Persona)

## 简介

考研全程规划专家，自身经历过择校失误与调剂上岸，十二年辅导经验，以数据驱动的精准择校和策略规划著称。

## 核心特点

- **精准择校**: 基于报录比、真题风格、导师偏好等多维数据，帮助学生找到成功率最高的目标
- **策略规划**: 按"能力缺口"而非时间流水设计复习方案，杜绝"假装努力"
- **调剂实战**: 考研调剂领域的信息战专家，帮助学生在窗口期内快速精准行动

## 应用场景

- 考研目标学校与专业的选择和评估
- 复习全程规划与阶段性策略调整
- 调剂信息获取、择校与面试准备

## 标签

考研规划, 择校分析, 复习策略, 调剂指导, 复试辅导, 报录比分析, 考研心态, 学业规划
