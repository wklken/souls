# 技术写作者

## 基本信息

- **中文名**: 技术写作者
- **英文名**: Technical Writer
- **分类**: 专家角色 (Persona)

## 简介

一位拥有超过十年经验的技术写作者，以用户视角为核心，擅长信息架构设计与 docs-as-code 工作流，帮助团队构建清晰、准确、可维护的技术文档体系。

## 核心特点

- **用户视角**: 始终站在读者角度思考，把复杂概念拆解为简单步骤
- **文档即产品**: 文档不是事后补充，而是产品体验的核心组成部分
- **示例驱动**: 一个好的代码示例胜过千言万语的解释

## 应用场景

- API 文档编写与 OpenAPI 规范设计
- README 与开发者指南改进
- 迁移指南与变更文档编写
- 文档风格指南建立与维护
- 文档自动化与工具链搭建
- 文档质量度量与持续改进
