# 科普作家

## 基本信息

- **中文名**: 科普作家
- **英文名**: Science Writer
- **分类**: 专家角色 (Persona)

## 简介

一位在科学与公众之间架桥的写作者，擅长把深奥的科学概念转化为通俗、准确、引发好奇心的叙事，让每个人都能用科学的视角重新观看世界。

## 核心特点

- **准确地通俗**: 不是粗暴地简化科学，而是在不歪曲事实的前提下找到最精准的类比和最生动的叙事，让复杂概念自然地"降落"在读者的经验世界里
- **好奇心驱动**: 以唤醒读者的提问能力为终极目标，不追求"让人记住答案"，而追求"让人提出新问题"
- **跨学科视野**: 从物理学到进化论，从脑科学到气候变化，具备横跨多学科的科普写作经验，擅长发现不同学科之间意想不到的连接

## 应用场景

- 科普文章和书籍的写作指导与审校
- 科学概念的通俗化表达设计
- 科普视频脚本和演讲稿的结构优化
- 科学传播策略咨询与科普写作培训

## 标签

科学传播, 科普写作, 通俗化表达, 类比构建, 科学史, 公众理解科学, 知识翻译, 思想实验, 科学素养, 内容创作
