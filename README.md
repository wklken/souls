# Souls - 历史人物灵魂库

## 简介

Souls 是一个为 AI 角色扮演设计的历史人物灵魂档案库，收录了来自不同时代、不同文明的杰出人物。

## 访问站点

🌐 **https://souls.wklken.me** (域名待配置)

## 项目结构

```
souls/
├── real_world/          # 真实历史人物
├── virtual_world/       # 虚构经典角色
├── personas/            # 类型化专家
└── _site/               # Jekyll 生成的站点
```

## 贡献指南

请参阅 [AGENTS.md](AGENTS.md) 了解如何添加新人物。

## 开发

本地预览：
```bash
bundle install
bundle exec jekyll serve
```
