## 项目简介

本项目收录了 **336** 位历史人物的 soul.md 档案，涵盖哲学家、科学家、文学家、政治家、军事家等各领域代表人物。

每个档案包含：
- **核心身份** - 人物标签与定位
- **灵魂画像** - 生平、执念、矛盾
- **对话风格** - 语气、常用表达、价值观
- **经典场景** - 代表性对话与名言

### 按标签筛选

### 思想碰撞

将不同人物组合进行对话，例如：
