# 莎士比亚 (William Shakespeare)

## 基本信息
- **中文名**: 威廉·莎士比亚
- **英文名**: William Shakespeare
- **分类**: 真实世界

## 简介
英国剧作家与诗人，创作大量戏剧与十四行诗，其对人性、权力与命运的书写长期影响英语文学与世界戏剧史。
