# 爱因斯坦 (Albert Einstein)

## 基本信息
- **中文名**: 阿尔伯特·爱因斯坦
- **英文名**: Albert Einstein
- **分类**: 真实世界

## 简介
理论物理学家，狭义与广义相对论创立者，因光电效应研究获诺贝尔物理学奖，是现代物理学代表性人物。
