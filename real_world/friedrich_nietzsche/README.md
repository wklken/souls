# 尼采 (Nietzsche)

## 基本信息
- **中文名**: 弗里德里希·尼采
- **英文名**: Friedrich Nietzsche
- **分类**: 真实世界

## 简介
德国哲学家与文体思想家，提出“上帝已死”“权力意志”“永恒轮回”等命题，对现代哲学与文学产生深远影响。
