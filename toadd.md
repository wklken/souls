# Souls 项目扩充计划 - 虚拟人物与专家角色

## 项目方向调整说明

基于当前政策环境，优先发展：
1. **虚拟人物 (virtual_world)** - 无版权风险，创意空间大
2. **专家角色 (personas)** - 实用价值高，可服务于特定场景

减少/暂停：在世历史人物（避免肖像权、名誉权争议）

---

## ⚠️ 版权风险声明

### 风险等级说明

| 等级 | 标识 | 说明 | 行动建议 |
|------|------|------|----------|
| 🟢 无风险 | 安全 | 公有领域或传统文化 | 可立即实施 |
| 🟡 低风险 | 注意 | 版权即将到期或存在争议 | 核实后实施 |
| 🟠 中等风险 | 谨慎 | 仍在版权保护期内 | 暂缓或寻找替代 |
| 🔴 高风险 | 避免 | 明确受版权/商标保护 | 不建议使用 |

### 各类别风险总览

| 类别 | 风险等级 | 说明 |
|------|----------|------|
| 中国古典文学（四大名著等） | 🟢 无风险 | 作者去世远超 50 年 |
| 中国神话传说 | 🟢 无风险 | 无作者，传统文化 |
| 希腊罗马神话 | 🟢 无风险 | 古代文化 |
| 莎士比亚作品 | 🟢 无风险 | 1616年去世 |
| 格林/安徒生童话 | 🟢 无风险 | 19世纪作品 |
| **金庸作品** | 🟠 中等风险 | 2018年去世，版权至2068年 |
| **古龙作品** | 🟠 中等风险 | 1985年去世，台湾版权至2035年 |
| 民国武侠小说 | 🟡 低风险 | 需核实具体作者去世时间 |
| 迪士尼改编形象 | 🔴 高风险 | 商标+版权双重保护 |
| 现代影视/游戏角色 | 🔴 高风险 | 严格版权保护 |

### 使用建议

1. **优先实施 🟢 无风险类别**
2. **🟠 中等风险类别暂缓**，待版权到期后再考虑
3. **避免使用任何影视改编版本的特定形象描述**，仅基于文字原作
4. **强化免责声明**：明确标注"基于公有领域文学作品"，与任何商业改编无关
